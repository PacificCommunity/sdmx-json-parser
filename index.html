SDMX-JSON Parser

SDMX-JSON Parser is a JavaScript library that parses SDMX-JSON data into a JSON object. It is designed to be used in web applications that need to consume SDMX-JSON data.

Latest version 0.2.0-beta.1 available on npm. API Documentation


        npm install sdmx-json-parser
